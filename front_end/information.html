Snake Game

Author: Sudikshya Bhattarai

This is a simple Snake game implemented using HTML, CSS, and JavaScript.

Usage Instructions

To play the game:

  • Use arrow keys to change the snake's direction.
  • Press the spacebar to pause or resume the game.
  • Eat apples to gain points and increase the snake's length.
  • Avoid collisions with obstacles and the snake itself.
